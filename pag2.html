subject:Defenishon html

Hypertext Markup Language (HTML) is the standard markup language for documents designed to be displayed in a web browser. It can be assisted by technologies such as Cascading Style Sheets (CSS is used to customize the style of the web site) and scripting languages such as JavaScript (JavaScript is used to make a dynamic web site) HTML elements are the building blocks of HTML pages, HTML elements are portrayed by tags and are written using angle brackets.
